Quick Start
===========

Prerequisite: Install <a href="http://nodejs.org/" target="_blank">Nodejs</a>

Setup the project:
> npm install  
> bower install  
> grunt build  

Launch the project
> npm start

And open a browser <a href="http://127.0.0.1:8080" target="_blank">http://127.0.0.1:8080</a>
